# Store_manager

#USED:
- JDK 8
- NetBeans
- MySQL Database System
- JavaFX
  
#Short Description:
It is a desktop app used to manage the entries and outputs of a store.

#FUNCTIONALITIES:
- Log-in/Log-out
- Add new products
- View Product
- Add daily sales
- View total amount of sales
- Calculate totaldaily sales
